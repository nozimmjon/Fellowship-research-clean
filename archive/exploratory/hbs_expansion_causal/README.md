# HBS Expansion Causal Module

This folder contains outputs for a separate exploratory HBS university-expansion design audit. It is intentionally isolated from the frozen LiTS main paper and from the older HBS supplementary-context workflow.

Expected contents:

- File and variable audits for the five annual HBS extracts.
- Design-specific linkage diagnostics for younger age windows.
- Harmonization and merge-readiness notes.
- Cohort-definition, outcome-maturity, and design-readiness outputs.
- Exploratory model tables only if the diagnostics justify estimation.

No file in this folder should be treated as a main-paper result unless it is explicitly migrated later after a separate design review.
