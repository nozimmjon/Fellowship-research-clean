# Raw Data

This cleaned repo does not duplicate the raw data from the legacy project.

The pipeline will use the first available source:

1. `FELLOWSHIP_RAW_DATA_ROOT`
2. this repo's `data/raw/`
3. sibling legacy repo `../Fellowship research/data/raw/`

To make this repo standalone, place raw files under:

- `data/raw/lits/`
- `data/raw/hbs/`
- `data/raw/admin/`
