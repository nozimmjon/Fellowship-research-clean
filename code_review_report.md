# Code Review Report: Intergenerational Educational Mobility in Uzbekistan

*Reviewed: 2026-04-07 | Language: R | Depth: main | Manuscript: reports/00_main.qmd*

## Overall Assessment

The project is well-engineered for a social science research pipeline. The `{targets}` DAG orchestrates 22 targets end-to-end, the main manuscript pulls every quantitative claim from pipeline-generated CSVs via inline R expressions, and all paths use `file.path()` with project-root discovery — no hardcoded absolute paths. The two principal issues are: (1) the policy brief contains ~30 hardcoded numbers that will silently diverge if the pipeline is re-run, and (2) the Module C logit specification drops the gender term pre-registered in Eq. 6 without documenting the deviation. A secondary concern is that Module A and Module B use different rank bases (years of schooling vs. ordinal score), which makes their "rank-rank slopes" not strictly comparable even though the manuscript compares them side by side.

## What's Working Well

- **Main manuscript is exemplary**: every quantitative claim in `00_main.qmd` uses accessor functions (`metric_est()`, `coef_est()`, `mech_est()`) reading from pipeline CSVs. Zero hardcoded numbers.
- **`{targets}` pipeline**: fully declared dependency graph with 22 targets covering all modules. `run_pipeline.R` regenerates everything.
- **`read_csv_safe()` with `stop()` on missing files**: rendering fails loudly rather than silently producing wrong numbers.
- **Empirical audit module** (`R/60_empirical_audit.R`): unusually thorough — generates inclusion flags, parent-measure robustness, missingness diagnostics, and a formal claim audit.
- **Education harmonization**: single locked mapping (`EDUCATION_LEVELS`, `UZB_REGION_HARMONIZATION_MAP`) with clear, defensive code across three LiTS waves.
- **Module C robustness**: five systematic scenarios with different split rules, weight usage, and parent proxies, with explicit support diagnostics.
- **Technical appendix and slides**: both properly source CSVs dynamically with graceful fallbacks.

## Code Quality Summary

### Pipeline Architecture
The `{targets}` DAG is clean and complete. All source files are loaded at the top of `_targets.R`. The `tar_option_set` uses `format = "rds"` throughout. The project root discovery function with environment variable override makes the project portable.

### Data Ingestion (R/20_ingest_data.R)
~570 lines but logically organized with clear function boundaries. The `education_level_to_years()` mapping (primary=6, lower_secondary=9, upper_secondary=11, tertiary=16) is a substantive research decision that should be documented with a citation.

### Module A (R/30_module_a_mobility.R, R/31_module_a_tier_a_descriptive.R)
Rank-rank slopes computed via `compute_rank_rank_slope()` using weighted OLS of `child_rank ~ parent_rank`. Ranks are based on `own_years_schooling` and `parent_years_schooling`. N<30 suppression enforced.

### Module B (R/40_module_b_determinants.R)
All five equation families implemented via `fixest::feols()` with `vcov = ~region`. Dynamic covariate selection via `is_covariate_informative()` with 20% non-missing threshold. Ranks are based on `own_ed_score` (ordinal 1-6), NOT years of schooling — this differs from Module A.

### Module C (R/50_module_c_mechanisms.R)
~560 lines. Uses `fixest::feglm(family="logit")` with `vcov = ~region`. Parent proxy switches to `mean_parent_years` (documented in manuscript). Five robustness scenarios. Module C minimum group N = 15 (vs. project-wide N = 30).

### Helpers (R/91_manuscript_helpers.R)
Clean accessor pattern. The slides redefine `fmt_pct` with `%%` symbol while the manuscript helper uses the word "percent" — formatting inconsistency.

### Paper Summary

- **Title**: Intergenerational Educational Mobility in Uzbekistan
- **Question**: How persistent is educational attainment across generations, how has persistence changed 2010-2022, and what household/regional factors correlate with mobility?
- **Data**: LiTS 2010, 2016, 2022-23 (repeated cross-sections); supplementary HBS 2021-2025
- **Sample**: Ages 25-64, weighted, N<30 suppression
- **Design**: Module A (descriptive), Module B (associational pooled models), Module C (pandemic mechanisms, bounded)
- **Key finding**: Rank-rank persistence rises 2010→2016, remains above 2010 in 2022-23 but 2016→2022 change is flat/imprecise
